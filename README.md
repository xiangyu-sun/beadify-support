# Beadify – Support

Bug reports and feature requests for **Beadify – Pixel Bead Art Maker** on the [App Store](https://apps.apple.com/app/id6744009407).

## File a report

- **Bug?** → [Open a bug report](../../issues/new?template=bug_report.yml)
- **Feature idea?** → [Open a feature request](../../issues/new?template=feature_request.yml)
- **Prefer email?** → alexander.xiangyu.sun@gmail.com *(no GitHub account needed)*

Before filing, search [existing issues](../../issues) — someone may have already reported it.

## Response time

This is an indie, one-person app. I read every report and respond as quickly as I can, typically within a few days.

## Privacy note

Issues are public. If your bug report includes a screenshot of a personal photo, please blur or crop it before attaching.
